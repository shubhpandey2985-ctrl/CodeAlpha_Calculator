# Calculator Program

A **menu-driven Calculator Program** developed in **C** as part of the **CodeAlpha C Programming Internship**. This project performs basic arithmetic operations using **switch-case**, **functions**, and **input validation**, demonstrating fundamental programming concepts in C.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- ❌ Division by Zero Validation
- 🔄 Menu-Driven Interface
- 🧩 Modular Function-Based Design
- 🚪 Exit Option

---

# 📋 Main Menu

```text
=============================
      CALCULATOR MENU
=============================
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit
=============================
Enter your choice:
```

---

# 📖 Program Flow (Flowchart)

```text
                START
                   │
                   ▼
        Display Calculator Menu
                   │
                   ▼
          Read User Choice
                   │
                   ▼
        Is Choice == Exit (5)?
             │            │
            No           Yes
             │            │
             ▼            ▼
     Enter Two Numbers   Exit Program
             │
             ▼
     Perform Selected Operation
             │
             ▼
        Display Result
             │
             ▼
      Return to Main Menu
```
